## üÜò TROUBLESHOOTING

### Problem: "Python not recognized"
**Solution:** Add Python to PATH or reinstall with PATH checkbox selected

### Problem: "Module not found"
**Solution:** Run `pip install -r requirements.txt`

### Problem: "MySQL connection error"
**Solution:** Check if MySQL service is running (XAMPP ‚Üí Start MySQL)

### Problem: "Streamlit not opening"
**Solution:** Try `python -m streamlit run streamlit_app.py`

### Problem: "Power BI import error"
**Solution:** Save CSV as UTF-8 encoding
